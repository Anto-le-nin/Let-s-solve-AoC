# Let-s-solve-AoC

This repo contain the code to solve the 4 firsts problem of Advent of Code

## Instruction

Get in the root of this repository.

To compile all the .c with the Makefile tip
`bash
make all
`

To execute the program of a day tip
`bash
make dayX
`

To erase the exec file tip 
`bash
make clean
`
